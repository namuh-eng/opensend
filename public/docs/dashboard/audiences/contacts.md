# Audience Contacts

Contacts are the people or addresses OpenSend knows about for audience, broadcast, topic, and preference workflows. The Contacts page lets you search, filter, add, inspect, and update contact records.

## Contact fields

Typical contact data includes email, first and last name, subscribed/unsubscribed state, custom properties, segments, topics, and timestamps. Use custom properties for data needed in segmentation or templates, but avoid storing secrets.

## Operational guidance

- Add contacts manually for support/testing or import through API workflows.
- Keep unsubscribed state authoritative for marketing mail.
- Use segments and topics instead of duplicating the same audience into many lists.
- Investigate suppressed addresses before re-subscribing a contact.
